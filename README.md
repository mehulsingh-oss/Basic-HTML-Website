# 🌐 Basic HTML Website

A simple multi-page website built using **only HTML5** as part of the **Roadmap.sh Basic HTML Website** project. This project focuses on learning semantic HTML, website structure, navigation, and forms without using CSS or JavaScript.

## 📖 About

The purpose of this project is to strengthen the fundamentals of HTML by creating a fully navigable multi-page website. It demonstrates the use of semantic elements, internal page linking, and accessible HTML structure.

## ✨ Features

* Built with pure HTML5
* Multi-page website
* Semantic HTML elements
* Navigation bar on every page
* Contact form
* Projects showcase
* Articles section
* Responsive-friendly HTML structure (ready for future CSS)
